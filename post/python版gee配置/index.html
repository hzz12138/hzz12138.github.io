1. Python环境配置

1.1 Python安装

Anaconda 是一个python的发行版，包括了python和很多常见的软件库, 和一个包管理器conda。装了Anaconda之后，就默认安了Python的很多包，而且也可以控制Python版本，后续比较方便。

  • 下载Anaconda

官网，不好访问的话，清华大学镜像库下载

  • 安装Anaconda

选择安装文件夹，建议在根目录下新建Anaconda文件夹

提示安装VScode，选择点击“skip”

安装完成。

1.2 配置Anaconda环境变量

此电脑——属性——高级系统设置——环境变量——path——编辑——新建

1
2
3
4

E:\Anaconda（Python需要）
E:\Anaconda\Scripts（conda自带脚本）
E:\Anaconda\Library\usr\bin
E:\Anaconda\Library\bin（jupyter notebook动态库）

1.3 检验是否安装成功

打开cmd，输入python和conda –version，若出现版本提示，则安装成功。

在开始菜单中找到anaconda navigator，若正常打开，则安装成功。

1.4 设置python国内镜像

打开Anaconda Prompt，依序运行以下代码：

1
2
3

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --set show_channel_urls yes
